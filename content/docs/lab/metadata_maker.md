# MetaDataMaker
This interface enables the generation, completion, and export of a structured, machine-readable JSON metadata file to accompany experimental datasets for downstream archiving and reuse.
By promoting consistent annotation of methods and data provenance, the tool supports reproducibility and alignment with FAIR (Findable, Accessible, Interoperable, Reusable) data principles.
The current implementation is a prototype tailored to patch-clamp experiments; however, ongoing work in our lab aims to extend it into a fully experiment-agnostic metadata framework.
